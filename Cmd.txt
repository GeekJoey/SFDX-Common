git init

git remote add origin git远程仓库位置（一次性配好，下次提交不用再配了）
git push origin master

打开当前文件夹

open .

sfdx force:org:create --definitionfile config/project-scratch-def.json --durationdays 30 --setalias SO环境别名 -v DevHub环境别名
